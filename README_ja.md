# AI GTM Developer

AIを使って、Googleタグマネージャーを使ったタグ実装の業務を自動化するためのプログラムやプロンプトをこのプロジェクトで構築していきます。

## 主な仕組み
GoogeタグマネージャーのAPIを用いて、現状のタグマネージャーに設定されているタグやトリガー、変数などのコンポーネントを取得し、JSON形式などで保存します。

その後AIを用いて、JSONファイルを更新し、タグの実装をJSONファイル上で行います。

すべてのタグ実装(JSONファイルの編集)が完了したら、再度APIを用いて、タグマネージャーに反映させます。

## アーキテクチャ
多くのユーザーが手軽に利用できるようにするために、できるだけ一般的なプログラミング言語と標準的なライブラリのみを用いて実装します。

- プログラミング言語: Python
